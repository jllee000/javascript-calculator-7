# javascript-calculator-precourse
## 프론트엔드 직군_이재림
**구현하고자 하는 기능**
1. 우선, @woowacourse/mission-utils 라이브러리 연결 후, console 확인
2. 기본 구분자(,;) 지정 -> 합 반환
3. 커스텀 구분자(//와 \n 사이) 지정 -> 합 반환
4. 로직 설명 주석으로 정리
5. 에러 분기처리 (입력값이 양수가 아니거나 구분자가 없을 경우)
   case(1) : 지정된 구분자나 숫자가 아닌 이외의 문자일때 (ex. a,ㄱ, 이외의 기호)
   case(2) : 숫자가 양수가 아닐 때
